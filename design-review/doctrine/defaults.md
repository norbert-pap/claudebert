# Opinionated Defaults

These are standing opinions. Apply them unless the project's DESIGN.md overrides.

- **Loading:** make it fast. If loading is needed, simpler is better. Exception: add fun if the wait is inevitable
- **Animation:** micro-interactions yes. Scroll-triggered parallax spectacles no
- **Dark mode:** nice-to-have, not essential
- **Desktop hamburger:** fine for non-essential items
- **Error copy:** don't make jokes if it might frustrate people. Context and brand dependent
- **Tables vs. cards:** depends on data type and quantity. If comparison is needed, table
- **Whitespace:** breathing room, not emptiness. Full-empty minimalism only for a narrow subset
- **Social proof:** essential for marketing sites
- **Fidelity:** spirit of the design over pixel-perfection
- **Footers:** sitemap. Don't overthink it
