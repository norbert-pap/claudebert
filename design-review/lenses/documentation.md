# Documentation / Content Site Lens

Apply when reviewing documentation sites, blogs, knowledge bases, wikis, or content-heavy sites.

## Evaluation Criteria

- **Reading experience:** measure (45-75 chars), line-height (1.5-1.7), contrast, comfortable font size (>=16px body)
- **Navigation and findability** (Covert) — can the user find what they're looking for? Is the information architecture intuitive?
- **Search that works.** If the site has more than 10 pages, search is essential, not optional
- **Code blocks readable and copyable?** Syntax highlighting, copy button, correct language detection

## Lens-Specific Provocations

- "Find a specific topic on this site. How many clicks? Did you use search or navigation? Which was faster?"
- "Read a full article. Is the reading experience comfortable, or did you want to widen/narrow the column?"
